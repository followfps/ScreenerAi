# Инструкция по использованию ScreenerAi

ScreanerAi — это умный органайзер ваших скриншотов на базе нейросети Qwen-VL.

## 🏃‍♂️ Как запустить

1. Запустите файл `ScreenerAi.exe`.
2. Приложение появится в системном трее (возле часов).

## ⚙️ Настройка

1. Нажмите правой кнопкой мыши на иконку в трее и выберите **Settings**.
2. **Root Directory**: Укажите папку, где лежат ваши категории (папки для сортировки).
3. **Qwen Session Token**: 
   - Зайдите на [chat.qwen.ai](https://chat.qwen.ai/).
   - Откройте консоль браузера (`F12` -> Console).
   - Выполните: `localStorage.getItem('token')`.
   - Скопируйте результат и вставьте в настройки ScreanerAi.
4. **Hotkey**: Нажмите на поле и задайте удобную комбинацию клавиш.
5. Нажмите **Save Changes**.

## 📸 Как пользоваться

1. Нажмите вашу горячую клавишу.
2. Выделите область экрана мышкой.
3. Дождитесь уведомления о сохранении. Файл автоматически попадет в нужную папку!
